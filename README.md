# dotfiles

各種設定ファイル集です。

## フォルダ構成

- `.vscode`: Visual Studio Codeの設定
- `autohotkey`: AutoHotkeyスクリプト
- `ctags.d`: Ctagsの設定
- `nvim`: Neovimの設定
- `scoop`: Scoopパッケージマネージャーの設定
- `vimrc`: Vimの設定
- `Windows Terminal`: Windows Terminalの設定

# Requirement

windowsユーザーです

# Usage

1. scoop自体のインストール
2. scoopを使って各種インストール
    scoop export > ファイル名でJsonファイル化
    scoop import Jsonファイルでインポート
3. シンボリックリンクは
    nvim
    ctags.d
    vimfiles
    の3点
4.

で環境構築が一瞬で終わるようにしたい
